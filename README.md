# 💣 TomTheBomber

## 🎮 About the Project

TomTheBomber is a game inspired by the classic **Bomberman** style gameplay. In this project, you navigate through levels, place bombs strategically, and try to eliminate enemies while avoiding getting caught in explosions yourself.

The game focuses on timing, positioning, and quick decision-making. Players must carefully plan where to place bombs to clear obstacles and defeat enemies, while also making sure they have a safe path to escape.
